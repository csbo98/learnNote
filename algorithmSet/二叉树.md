# 二叉树

二叉树相关题目的解题套路相对简单，几乎所有二叉树相关的题目都是用**前中后序遍历、层次遍历**这四个固定的模版。
解题时，需要做的仅仅是**定义好递归函数的含义**。

## 广度优先遍历(层次遍历)

```C++
        // 初始化队列：创建队列，并向其中加入第一个元素
        queue<TreeNode*> que;
        que.push(root);

        while(!que.empty()) {
            // 这是一个按照层对树中结点进行处理的方法，在需要对树按照层次
            // 处理时很有用。
            int size = que.size();

            for(int i = 0; i < size; i++) {
                TreeNode *node = que.front();
                que.pop();

                // 对当前的结点做处理；

                // 把左右子结点加入队列
                if(node->left) {
                    que.push(node->left);
                }
                if(node->right) {
                    que.push(node->right);
                }
            }
        }
```
